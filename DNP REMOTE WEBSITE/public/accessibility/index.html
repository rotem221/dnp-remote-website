Accessibility Statement

Last updated: March 2026

1. Our Commitment

DNP Remote is committed to making its website accessible to everyone, including people with disabilities. We strive to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standard.

We believe that every developer deserves equal access to digital tools, and we continuously work to improve the accessibility of this website.

2. Accessibility Standards

This website is built and maintained in accordance with:

  • WCAG 2.1 Level AA - Web Content Accessibility Guidelines published by the W3C

We follow the four core principles of accessibility:

  • Perceivable - content is presented in ways that can be perceived by all users
  • Operable - navigation and interactions are possible via multiple input methods
  • Understandable - content and UI are clear and predictable
  • Robust - content works reliably with current and future assistive technologies

3. Accessibility Features on This Website

3.1 Navigation and structure

  • Clear heading hierarchy (H1–H6) for quick page navigation
  • Full keyboard navigation support throughout all pages
  • Logical and consistent tab order
  • Navigation landmarks and ARIA labels on interactive elements
  • Descriptive link text with clear purpose

3.2 Content and presentation

  • Alt text on all meaningful images
  • Color contrast ratio of at least 4.5:1 for body text (WCAG AA)
  • Text can be resized up to 200% without loss of content or functionality
  • Readable fonts at all screen sizes
  • Color is not the sole means of conveying information

3.3 Assistive technology support

  • Compatible with VoiceOver (iOS / macOS) and NVDA / JAWS (Windows)
  • ARIA roles and labels on dynamic elements
  • Support for high-contrast OS mode
  • Support for screen magnification

4. Built-in Accessibility Widget

Every page on this website includes a built-in accessibility panel, accessible via the accessibility icon on the left side of the screen. It offers:

  • Text size adjustment (three levels)
  • High contrast mode
  • Grayscale mode
  • Link highlighting
  • Readable font (dyslexia-friendly typeface)
  • Increased text spacing
  • Pause animations
  • Large cursor
  • Reading guide line

Your preferences are saved automatically and will be applied on future visits.

5. Known Limitations

Despite our best efforts, some areas may not yet be fully accessible:

  • Third-party embeds: Content from external services (e.g., App Store badges) may not fully comply with WCAG standards
  • Older content: Some older marketing assets are being progressively updated

We prioritise fixes based on user impact and update this statement as improvements are made.

6. Accessibility Testing

We conduct ongoing accessibility testing including:

  • Automated checks: Using Axe, WAVE, and Google Lighthouse
  • Manual reviews: Keyboard-only navigation, screen reader testing with VoiceOver and NVDA
  • Periodic audits: Reviewed with each major site update
