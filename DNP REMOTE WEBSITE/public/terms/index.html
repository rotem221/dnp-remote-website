2. Description of the Service

DNP Remote is an iOS application that connects your iPhone to a Mac running the Mac Bridge companion server, allowing you to remotely control AI coding agents such as Claude Code, Cursor, VS Code, and OpenAI Codex. Features include:

  • Chat interface with streaming AI agent responses
  • File browser - browse and view project files on your Mac
  • Screen mirroring - view your simulator or Mac screen from your iPhone
  • Session history with token usage tracking
  • Connection via local network (Bonjour auto-discovery) or Tailscale VPN
  • QR code pairing for instant setup
